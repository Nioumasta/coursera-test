# My Cheer Bowtique

Site Next.js 14 multilingue (FR/EN) optimisé pour le SEO européen, les conversions e-commerce et les Core Web Vitals.

## Scripts
- `npm run dev` : démarrage local
- `npm run build` : build production
- `npm start` : serveur production
- `npm run lint` : vérification lint

## Points clés
- App Router avec sous-répertoires `/fr` et `/en`
- TailwindCSS + design system holographique
- Métadonnées dynamiques + hreflang + JSON-LD (Organization, Product, FAQ, Breadcrumb, ItemList)
- Sitemap et robots automatiques
- Scripts marketing préconfigurés (GA4, GTM, Meta, Pinterest, Hotjar, Clarity, Mailchimp, OneSignal)
- Pages silo (catalogue, produit, personnalisation, clubs, équipes, blog, FAQ, politiques)
