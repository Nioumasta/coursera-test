# Checklist SEO + CRO

- [ ] Titres H1/H2 localisés pour chaque langue.
- [ ] Métadonnées dynamiques et hreflang FR/EN activés.
- [ ] JSON-LD Organization, Product, FAQ, Breadcrumb, ItemList actifs.
- [ ] CTAs visibles en mobile, sticky header fonctionnel.
- [ ] Urgences stock + preuves sociales sur les pages produit.
- [ ] Images compressées WebP/AVIF, lazy-loading natif.
- [ ] Vitesse vérifiée (LCP < 2.5s, CLS < 0.1, INP < 200ms).
- [ ] Scripts marketing chargés en `afterInteractive` pour limiter le coût initial.
- [ ] Sitemap et robots générés automatiquement après build.
- [ ] Tests d’accessibilité (contraste, aria-labels) passés.
