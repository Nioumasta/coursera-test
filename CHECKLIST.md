# SEO + CRO checklist
- Titres et meta descriptions localisées par page.
- Hreflang /fr /en + x-default.
- Données structurées Organization, BreadcrumbList, Product, FAQPage.
- Vitesse : images WebP/AVIF, lazy-loading, Tailwind purgé.
- Core Web Vitals : CLS réduit (dimensions images), fonts système.
- CRO : CTA sticky, preuve sociale, urgence stock, upsell cross-sell.
- Tracking : GA4, GTM, Meta Pixel, Pinterest, Hotjar, Clarity, Mailchimp/Klaviyo, OneSignal, Trustpilot/Yotpo.
- Sitemap & robots mis à jour après ajout de pages.
- Accessibilité : contrastes, aria-label, navigation clavier.
